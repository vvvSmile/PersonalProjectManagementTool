快捷键：
Alt + Enter   引入类
Alt + Insert    set/get; 构造方法;  toString; 重写方法。。。
Ctrl + Alt + b 查看接口实现类中方法（就是我们使用接口编程时，在调用实现类方法处直接Ctrl+鼠标左键方法，只能进入到接口中定义的方法处，但是使用该快捷键则直接到实现类的该方法中了）
Ctrl+Alt+T 将代码包在一个块中，例如try/catch  ;synchronized等
CTRL+ALT+I 自动缩进
